## 1. วิธีเปิดหน้าสั่งสินค้า

เปิดแอป **LINE** บนมือถือ  
จากนั้นกดลิงก์หรือแบนเนอร์ที่ทางร้าน T&Y Noodle ส่งให้ใน LINE  
ระบบจะเปิดหน้าสั่งสินค้าขึ้นมาโดยอัตโนมัติ **ภายใน LINE** เลย ไม่ต้องออกไปใช้แอปอื่น

---

## 2. เข้าสู่ระบบด้วย LINE

เมื่อเปิดหน้าสั่งสินค้าครั้งแรก จะเจอหน้าดังนี้:

```
┌─────────────────────────────────┐
│                                 │
│   [ โลโก้ T&Y Noodle ]          │
│                                 │
│   เส้นรังนก T&Y Noodle          │
│   กรุณากดเข้าสู่ระบบด้วย LINE   │
│   เพื่อสั่งสินค้า               │
│                                 │
│  ┌───────────────────────────┐  │
│  │  [ ไอคอน LINE ]           │  │
│  │  เข้าสู่ระบบด้วย LINE     │  │
│  └───────────────────────────┘  │
│                                 │
└─────────────────────────────────┘
```

**ขั้นตอน:**

1. กด **ปุ่มสีเขียว "เข้าสู่ระบบด้วย LINE"** ตรงกลางหน้าจอ
2. LINE จะถามว่า "อนุญาตให้แอปนี้ใช้ข้อมูล LINE ของคุณหรือไม่?" — กด **อนุญาต** หรือ **ยืนยัน**
3. ระบบจะตรวจสอบข้อมูลอัตโนมัติ (อาจใช้เวลา 2–3 วินาที)
   - ถ้าเคยลงทะเบียนไว้แล้ว → จะเข้าสู่ **หน้าเลือกสินค้า** ทันที
   - ถ้ายังไม่เคยลงทะเบียน → จะไปยัง **ขั้นตอนลงทะเบียน** (ข้อ 3)

> **หมายเหตุ:** ถ้ากดแล้วไม่มีอะไรเกิดขึ้น ลองปิดหน้าต่างแล้วเปิดลิงก์ใหม่อีกครั้ง

## 3. ลงทะเบียนครั้งแรก

หน้านี้จะถามว่า **"คุณเคยเป็นลูกค้า T&Y Noodle มาก่อนหรือเปล่า?"**

```
┌─────────────────────────────────┐
│   [ ไอคอนร้านค้า ]              │
│   ยินดีต้อนรับ                  │
│   คุณเคยเป็นลูกค้า T&Y Noodle  │
│   มาก่อนหรือเปล่า?              │
│                                 │
│  ┌───────────────────────────┐  │
│  │ 🏪 ลูกค้าเก่า              │  │
│  │    เคยสั่งกับ T&Y Noodle  │  │
│  │    มาก่อน          ▼      │  │
│  └───────────────────────────┘  │
│                                 │
│  ┌───────────────────────────┐  │
│  │ 👤 ลูกค้าใหม่              │  │
│  │    ยังไม่เคยสั่งสินค้า  → │  │
│  └───────────────────────────┘  │
│                                 │
└─────────────────────────────────┘
```

**เลือกตามสถานการณ์:**
- กด **"ลูกค้าเก่า"** → ถ้าเคยซื้อสินค้ากับ T&Y Noodle มาแล้ว
- กด **"ลูกค้าใหม่"** → ถ้ายังไม่เคยซื้อเลย

### 3.1 ลูกค้าเก่า (เคยซื้อสินค้ากับ T&Y Noodle มาก่อน)

เมื่อกด **"ลูกค้าเก่า"** จะมีแบบฟอร์มกางออกมาให้กรอก:

```
┌─────────────────────────────────┐
│  กรอกข้อมูลร้านค้า              │
│                                 │
│  ชื่อร้านค้า *                  │
│  [ ชื่อร้านของคุณ............. ]│
│                                 │
│  เบอร์โทรศัพท์ (ถ้ามี)          │
│  [ 0xx-xxx-xxxx............... ]│
│                                 │
│  ที่อยู่ตลาดในการจัดส่งสินค้า   │
│  (บ้านเลขที่ / ถนน / ซอย)       │
│  [ ........................... ]│
│                                 │
│  จังหวัด *                      │
│  [ — เลือกจังหวัด — ▼ ]         │
│                                 │
│  อำเภอ / เขต *                  │
│  [ — เลือกอำเภอ/เขต — ▼ ]       │
│                                 │
│  ตำบล / แขวง *                  │
│  [ — เลือกตำบล/แขวง — ▼ ]       │
│                                 │
│  รหัสไปรษณีย์ (ระบบกรอกให้เอง) │
│                                 │
│  [ ✓ ยืนยันข้อมูลร้านค้า ]      │
└─────────────────────────────────┘
```

**วิธีกรอกข้อมูล ทีละช่อง:**

**ช่องที่ 1 — ชื่อร้านค้า (จำเป็น ⭐)**
- กดที่ช่องสี่เหลี่ยม แล้วพิมพ์ **ชื่อร้านค้าของคุณ**
- เช่น "ร้านก๋วยเตี๋ยวแม่สมศรี" หรือ "ตลาดนัดหนองแขม"

**ช่องที่ 2 — เบอร์โทรศัพท์ (ไม่บังคับ)**
- กรอกหรือไม่กรอกก็ได้
- ถ้ากรอก ให้พิมพ์เบอร์มือถือ เช่น 0812345678

**ช่องที่ 3 — ที่อยู่ตลาด (ไม่บังคับ)**
- กรอกบ้านเลขที่ หมู่บ้าน ถนน ซอย
- เช่น "15/3 ตลาดนัดสุขุมวิท ซ.22"

**ช่องที่ 4 — จังหวัด (จำเป็น ⭐)**
1. กดที่ช่อง **"— เลือกจังหวัด —"**
2. รายชื่อจังหวัดจะปรากฏขึ้น
3. เลื่อนหาจังหวัดที่อยู่แล้วกดเลือก

**ช่องที่ 5 — อำเภอ/เขต (จำเป็น ⭐)**
1. กดที่ช่อง **"— เลือกอำเภอ/เขต —"**
2. รายชื่ออำเภอในจังหวัดที่เลือกจะปรากฏขึ้น
3. กดเลือกอำเภอของคุณ

**ช่องที่ 6 — ตำบล/แขวง (จำเป็น ⭐)**
1. กดที่ช่อง **"— เลือกตำบล/แขวง —"**
2. กดเลือกตำบลของคุณ
3. **รหัสไปรษณีย์จะปรากฏขึ้นเองอัตโนมัติ** ไม่ต้องกรอก

**เมื่อกรอกครบแล้ว:**
- กดปุ่ม **"✓ ยืนยันข้อมูลร้านค้า"** (ปุ่มสีน้ำเงินเข้มด้านล่าง)
- รอสักครู่ — ระบบกำลังบันทึกข้อมูล
- เมื่อเสร็จ จะเข้าสู่ **หน้าเลือกสินค้า** โดยอัตโนมัติ

> **⚠️ สำคัญ:** ถ้าระบบแจ้งข้อผิดพลาด (เช่น "กรุณากรอกชื่อร้านค้า") ให้ตรวจสอบช่องที่มีเครื่องหมาย ⭐ ว่ากรอกครบหรือยัง

---

### 3.2 ลูกค้าใหม่ (ยังไม่เคยซื้อ)

เมื่อกด **"ลูกค้าใหม่"** จะเจอหน้าสมัครสมาชิก:

```
┌─────────────────────────────────┐
│  ← กลับ                         │
│                                 │
│  [ ไอคอนสีเขียว ]               │
│  สมัครเป็นลูกค้าใหม่            │
│  ฝากชื่อและเบอร์ไว้             │
│  เราจะติดต่อกลับโดยด่วน         │
│                                 │
│  ชื่อ - นามสกุล *               │
│  [ ชื่อของคุณ................. ]│
│                                 │
│  เบอร์โทรศัพท์ *                │
│  [ 0xx-xxx-xxxx............... ]│
│                                 │
│  [ 👤 ส่งข้อมูลให้ทีมงาน ]      │
│                                 │
│  ┌───────────────────────────┐  │
│  │ 📞 สามารถติดต่อเราได้ที่  │  │
│  │    081-903-4686           │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

**วิธีกรอก:**

1. กดช่อง **"ชื่อ - นามสกุล"** แล้วพิมพ์ชื่อจริงของคุณ
2. กดช่อง **"เบอร์โทรศัพท์"** แล้วพิมพ์เบอร์ที่ติดต่อได้
3. กดปุ่ม **"ส่งข้อมูลให้ทีมงาน"**

ระบบจะส่งข้อมูลของคุณให้ทีม T&Y Noodle และทางร้านจะโทรติดต่อกลับหาคุณ

**หรือถ้าต้องการโทรหาทางร้านเลย:**
- กดที่แถบ **"📞 081-903-4686"** เพื่อโทรหาทีมงานได้ทันที

---

**หน้าหลังส่งข้อมูลสำเร็จ (ลูกค้าใหม่)**

หลังจากกดส่งข้อมูล จะเจอหน้านี้:

```
┌─────────────────────────────────┐
│                                 │
│       [ ✓ ไอคอนสีเขียว ]       │
│                                 │
│    ส่งข้อมูลเรียบร้อย!          │
│    ทีมงาน T&Y Noodle            │
│    ได้รับข้อมูลของคุณแล้ว       │
│    เราจะติดต่อกลับหาคุณโดยด่วน  │
│                                 │
│  ┌───────────────────────────┐  │
│  │ 📞 หรือโทรหาเราได้เลย    │  │
│  │    081-903-4686           │  │
│  └───────────────────────────┘  │
│                                 │
│  [ กลับไปยัง LINE ]             │
│                                 │
└─────────────────────────────────┘
```

- กด **"กลับไปยัง LINE"** เพื่อปิดหน้าต่าง
- รอทางร้านโทรกลับ — ท่านจะได้รับการติดต่อเร็วๆ นี้

## 4. หน้าเลือกสินค้า (หน้าหลัก)

เมื่อเข้าระบบสำเร็จ จะเจอหน้าเลือกสินค้า ซึ่งแบ่งเป็นส่วนต่างๆ ดังนี้:

```
┌─────────────────────────────────────┐
│  [ ภาพแบนเนอร์ T&Y Noodle ]         │
│                   [ 🛒 ตะกร้า (0) ] │
│                                     │
│         [ รูปโปรไฟล์ LINE ]          │
│           ชื่อร้านค้าของคุณ          │
│             ชื่อ LINE ของคุณ         │
│                                     │
│  ┌──────────────────────────────┐   │
│  │ 🕐 เปิดรับออเดอร์อยู่        │   │
│  │    รับออเดอร์ถึง 17.00 น.    │   │
│  │                  [ เปิด 🟢 ] │   │
│  └──────────────────────────────┘   │
│                                     │
│  [🔍 ค้นหาสินค้า...              ]  │
│                                     │
│  [ ทั้งหมด ] [ หมวด A ] [ หมวด B ] │
│                                     │
│  ─── [ 📦 สินค้า ] [⭐ โปรด] [🕐 ล่าสุด] ───
│                                     │
│  สินค้าที่สั่งซื้อบ่อย             │
│  [ สินค้า A ► ] [ สินค้า B ► ]     │
│                                     │
│  ┌──────┐ ┌──────┐ ┌──────┐        │
│  │รูป   │ │รูป   │ │รูป   │        │
│  │สินค้า│ │สินค้า│ │สินค้า│        │
│  │ชื่อ  │ │ชื่อ  │ │ชื่อ  │        │
│  │    ➕│ │    ➕│ │    ➕│        │
│  └──────┘ └──────┘ └──────┘        │
└─────────────────────────────────────┘
```

### แถบสถานะรับออเดอร์

แถบนี้อยู่ด้านบนของหน้า บอกให้รู้ว่าตอนนี้สั่งสินค้าได้หรือยัง:

| แถบสี | ความหมาย |
|-------|----------|
| 🟢 **สีน้ำเงิน "เปิดรับออเดอร์อยู่"** | สั่งสินค้าได้เลย ก่อน 17:00 น. |
| 🔴 **สีส้ม "ปิดรับออเดอร์แล้ว"** | เลยเวลา 17:00 น. แล้ว รอเปิดใหม่เที่ยงคืน |

> ระบบรับออเดอร์ **ตั้งแต่เที่ยงคืน (00:00 น.) จนถึง 17:00 น.** ของทุกวัน

### ปุ่มตะกร้าสินค้า (มุมบนขวา)

```
     [ 🛒 ]
      (3)   ← ตัวเลขสีแดง = จำนวนรายการที่เลือกแล้ว
```

- ปุ่มรูปรถเข็น อยู่มุมบนขวาของหน้า
- **ตัวเลขสีแดงบนปุ่ม** = จำนวนรายการสินค้าที่เพิ่มเข้าตะกร้าแล้ว
- กดปุ่มนี้เพื่อดูสินค้าที่เลือกไว้และยืนยันการสั่งซื้อ

---

### ช่องค้นหาสินค้า

```
[ 🔍 ค้นหาสินค้า...              ✕ ]
```

- กดที่ช่องแล้วพิมพ์ชื่อสินค้าที่ต้องการ เช่น "เส้น" หรือ "บะหมี่"
- สินค้าที่ตรงกับคำค้นหาจะแสดงทันที
- กดปุ่ม **✕** (กากบาทด้านขวา) เพื่อล้างคำค้นหา

---

### ปุ่มหมวดหมู่สินค้า (Category Pills)

```
[ ทั้งหมด ] [ เส้น ] [ ซอส ] [ บรรจุภัณฑ์ ]
```

- แถวปุ่มกลมๆ อยู่ใต้ช่องค้นหา
- กด **"ทั้งหมด"** เพื่อแสดงสินค้าทุกชนิด
- กดหมวดหมู่อื่นๆ เพื่อกรองดูเฉพาะสินค้าในหมวดนั้น
- **ปุ่มที่กดเลือกอยู่จะเป็นสีน้ำเงินเข้ม**

---

### แท็บแสดงสินค้า (3 แท็บ)

```
[ 📦 สินค้า ] | [⭐ รายการโปรด] | [🕐 ล่าสุด]
```

อยู่ใต้ปุ่มหมวดหมู่ มี 3 แท็บ:

| แท็บ | หน้าที่ |
|------|---------|
| **📦 สินค้า** | แสดงสินค้าทั้งหมด (ค่าเริ่มต้น) |
| **⭐ รายการโปรด** | แสดงเฉพาะสินค้าที่คุณกดดาวไว้ |
| **🕐 ล่าสุด** | แสดงสินค้าจากออเดอร์ล่าสุดของคุณ |

กดแท็บที่ต้องการได้เลย **เส้นใต้สีน้ำเงินจะเลื่อนตามแท็บที่เลือก**

### สินค้าที่สั่งซื้อบ่อย

```
┌──────────────────────────────────────────────┐
│ ┌────────────┐  ┌────────────┐               │

│ │  [รูปสินค้า]  │  │  [รูปสินค้า]  │               │
│ │ ชื่อสินค้า │  │ ชื่อสินค้า │               │
│ │ หน่วย กก.  │  │ หน่วย ถุง  │               │
│ │          ➕│  │          ➕│               │
│ └────────────┘  └────────────┘               │
└──────────────────────────────────────────────┘
```

- แถวแสดงสินค้าที่คุณสั่งบ่อยที่สุด
- **เลื่อนซ้าย-ขวา** เพื่อดูสินค้าทั้งหมดในแถว
- กด **ปุ่ม ➕** เพื่อเพิ่มสินค้าเข้าตะกร้าได้เลยทันที (จำนวนขั้นต่ำ)

---

### การ์ดสินค้า

```
┌──────────────┐
│  รูปสินค้า   │
│  [ (จำนวน) ] │  ← ตัวเลขน้ำเงินมุมซ้ายบน = เพิ่มในตะกร้าแล้ว
│              │
│           ⭐ │  ← ปุ่มดาว มุมขวาบน
│              │
│ ชื่อสินค้า  │
│          ➕  │  ← ปุ่มบวก สีน้ำเงิน
└──────────────┘
```

**ส่วนต่างๆ บนการ์ดสินค้า:**

- **รูปสินค้า** — กดที่รูปหรือชื่อสินค้าเพื่อดูรายละเอียดเพิ่มเติม
- **ตัวเลขสีน้ำเงิน** (มุมซ้ายบน) — แสดงว่าคุณเพิ่มสินค้านี้เข้าตะกร้าแล้วกี่หน่วย
- **⭐ ดาว** (มุมขวาบน) — กดเพื่อ "บันทึกรายการโปรด" สีเหลือง = บันทึกแล้ว, สีเทา = ยังไม่บันทึก
- **➕ ปุ่มบวก** (ขวาล่าง) — กดเพื่อเปิดหน้ารายละเอียดและเพิ่มเข้าตะกร้า

---

## 5. การดูรายละเอียดสินค้า

เมื่อกดที่รูปสินค้าหรือปุ่ม ➕ จะมีหน้าต่างรายละเอียดสินค้าเลื่อนขึ้นมาจากด้านล่าง:

```
┌─────────────────────────────────────────────────────┐
│                                               [ ✕ ] │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │                                             │   │
│  │             รูปสินค้า (ใหญ่)                │   │
│  │      ◀                             ▶       │   │
│  │  [ ●  ○  ○ ] ← จุดบอกว่ามีกี่รูป            │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  ชื่อสินค้า (ตัวหนา)                               │
│                                                     │
│  ┌──────────────────────────────────────────────┐  │
│  │  หน่วย: กก.   |   หน่วย: ถุง                 │  │
│  │  [ เลือก ]    |   [ เลือก ]                 │  │
│  └──────────────────────────────────────────────┘  │
│                                                     │
│  สินค้าแนะนำอื่นๆ                                   │
│  [รูป]  [รูป]  [รูป]  [รูป]  →                     │
│                                                     │
│─────────────────────────────────────────────────── │
│  [ ➖  0  กก.  ➕ ]     [ 🛒 เพิ่มเข้าตะกร้า ]     │
└─────────────────────────────────────────────────────┘
```

**ส่วนต่างๆ ในหน้ารายละเอียดสินค้า:**

### รูปภาพสินค้า

- แสดงรูปสินค้าขนาดใหญ่
- ถ้ามีหลายรูป สามารถ **ปัดซ้าย-ขวา** หรือกดปุ่ม **◀ ▶** เพื่อดูรูปอื่นๆ ได้
- จุดกลมเล็กๆ ด้านล่างรูป บอกว่าอยู่ที่รูปที่เท่าไหร่จากทั้งหมดกี่รูป

### ตัวเลือกหน่วยสินค้า

- บางสินค้ามีหลายหน่วย เช่น "กก." หรือ "ถุง"
- กดเลือกหน่วยที่ต้องการก่อน แล้วค่อยตั้งจำนวน

### การตั้งจำนวนสินค้า (ด้านล่างสุด)

```
[ ➖ ] [ 0 ] [ กก. ] [ ➕ ]     [ 🛒 เพิ่มเข้าตะกร้า ]
```

- **ปุ่ม ➕** (สีน้ำเงิน ด้านขวา) — เพิ่มจำนวนสินค้าครั้งละ 1 หน่วย (หรือตามหน่วยขั้นต่ำ)
- **ปุ่ม ➖** (ด้านซ้าย) — ลดจำนวนสินค้า
- **ตัวเลขตรงกลาง** — จำนวนที่คุณตั้งไว้
- กดปุ่ม **➕** จนได้จำนวนที่ต้องการ แล้วกด **"🛒 เพิ่มเข้าตะกร้า"**

> ถ้าปุ่ม "เพิ่มเข้าตะกร้า" ยังเป็นสีเทาอยู่ — ให้กดปุ่ม ➕ ก่อนเพื่อเลือกจำนวน

### ปุ่มปิดหน้าต่าง

- กดปุ่ม **✕** มุมขวาบน หรือ **ลากหน้าต่างลง** เพื่อปิดและกลับไปหน้าสินค้า

## 6. ตะกร้าสินค้า และการยืนยันคำสั่งซื้อ

เมื่อเลือกสินค้าครบแล้ว กดปุ่ม **🛒** (ตะกร้า) มุมบนขวาของหน้าหลัก

```
┌─────────────────────────────────────┐
│  ← กลับ                             │
│          รายการที่เลือก             │
│                                     │
│  ┌───────────────────────────────┐  │
│  │ [รูป] ชื่อสินค้า A            │  │
│  │       หน่วย: กก.              │  │
│  │       [ ➖ ]  5  [ ➕ ]  [🗑️] │  │
│  └───────────────────────────────┘  │
│                                     │
│  ┌───────────────────────────────┐  │
│  │ [รูป] ชื่อสินค้า B            │  │
│  │       หน่วย: ถุง              │  │
│  │       [ ➖ ]  3  [ ➕ ]  [🗑️] │  │
│  └───────────────────────────────┘  │
│                                     │
│  ─────────────────────────────────  │
│                                     │
│  [ ✓ ยืนยันสั่งซื้อ ]              │
│                                     │
└─────────────────────────────────────┘
```

**ส่วนต่างๆ ในหน้าตะกร้า:**

### รายการสินค้าที่เลือก

แต่ละรายการจะแสดง:
- **รูปสินค้า** — ขนาดเล็กด้านซ้าย
- **ชื่อสินค้า** และ **หน่วย**
- **ปุ่ม ➖ ➕** — ปรับเพิ่ม/ลดจำนวน
- **ปุ่ม 🗑️** (ถังขยะ) — ลบสินค้ารายการนี้ออกจากตะกร้า

### ปรับจำนวนสินค้าในตะกร้า

- กด **➕** เพิ่มจำนวน
- กด **➖** ลดจำนวน (ถ้าลดจนเหลือ 0 สินค้าจะหายไปจากตะกร้า)

### กลับไปเพิ่มสินค้า

กดปุ่ม **← กลับ** มุมซ้ายบน เพื่อกลับไปหน้าเลือกสินค้าและเพิ่มสินค้าเพิ่มเติมได้

### ยืนยันการสั่งซื้อ

เมื่อตรวจสอบรายการครบแล้ว กด **"✓ ยืนยันสั่งซื้อ"** (ปุ่มใหญ่สีน้ำเงินด้านล่าง)

> ระบบจะส่งออเดอร์ไปที่ร้าน T&Y Noodle ทันที

## 9. คำถามที่พบบ่อย

---

**❓ กดปุ่ม "เข้าสู่ระบบด้วย LINE" แล้วไม่มีอะไรเกิดขึ้น**

→ เวลารับออเดอร์คือ **00:00 น. – 17:00 น.** เท่านั้น
