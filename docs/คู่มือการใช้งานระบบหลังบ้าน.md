## 1. การเข้าสู่ระบบ

### 🔑 หน้า Login คืออะไร?
หน้าแรกที่เปิดขึ้นมาก่อนเข้าใช้งานระบบ จะต้องกรอก **รหัส PIN 6 หลัก** เพื่อยืนยันตัวตน

### วิธีเข้าสู่ระบบ
1. เปิดเบราว์เซอร์ (เช่น Chrome, Safari) แล้วพิมพ์ลิงก์ของระบบ
2. จะเห็นหน้าให้กรอก PIN — **กรอกตัวเลข 6 หลัก** ที่ได้รับจากผู้ดูแลระบบ
3. กด **Enter** หรือรอระบบตรวจสอบอัตโนมัติ
4. ถ้า PIN ถูกต้อง → จะเข้าสู่หน้าแดชบอร์ดทันที

### ข้อผิดพลาดที่อาจเจอ
| ข้อความ | ความหมาย | วิธีแก้ |
|---------|----------|---------|
| "PIN ไม่ถูกต้อง" | กรอกรหัสผิด | ลองกรอกใหม่ให้ถูกต้อง |
| "PIN ถูกล็อก" | กรอกผิดหลายครั้งเกินไป | ติดต่อผู้ดูแลระบบ |

### บทบาทผู้ใช้งาน (Role)
ระบบมี **3 ระดับ** — แต่ละ PIN จะได้รับสิทธิ์ต่างกัน:
- **Admin (ผู้ดูแล)** — เข้าถึงได้ทุกหน้า รวมถึงสร้าง/แก้ไข/ลบข้อมูลทุกอย่าง
- **Member (สมาชิก)** — เข้าได้แค่แดชบอร์ดและรายงาน ดูได้อย่างเดียว แก้ไขไม่ได้
- **Warehouse (คลังสินค้า)** — เข้าได้เฉพาะหน้าใบจัดส่งและใบจัดของ ไม่มีเมนูด้านข้าง

## 2. เมนูด้านข้าง

### 📱 เมนูอยู่ที่ไหน?
บนมือถือ → กดปุ่ม **☰ (เส้น 3 เส้น)** มุมซ้ายบน เมนูจะเลื่อนออกมาจากด้านซ้าย  
บนคอมพิวเตอร์ → เมนูจะแสดงตลอดที่ด้านซ้ายมือ

### รายการเมนูทั้งหมด

```
🏠 แดชบอร์ด
📦 สต็อก
📋 รายการออเดอร์
📊 สรุปออเดอร์
🚚 ใบจัดส่ง
💰 ใบวางบิล

📈 รายงาน (กดเพื่อขยาย)
   ├── ภาพรวมยอดขายรายปี
   ├── ยอดขายสินค้า
   ├── ยอดขายตามร้านค้า
   └── รายงานใบจัดส่ง

⚙️ ตั้งค่า (กดเพื่อขยาย)
   ├── จัดการสินค้า
   ├── จัดการร้านค้า
   └── จัดการรถ

🚪 ออกจากระบบ
```

> **วิธีกดเมนู "รายงาน" และ "ตั้งค่า":**  
> กดที่คำว่า "รายงาน" หรือ "ตั้งค่า" → เมนูย่อยจะขยายออกมา กดอีกครั้งเพื่อปิด

## 3. แดชบอร์ด

### 🏠 หน้านี้คืออะไร?
หน้าหลักของระบบ ใช้ดู**ภาพรวมของวันนี้** เช่น มีออเดอร์กี่ใบ ยอดขายเท่าไหร่ มีอะไรที่ต้องทำบ้าง

### สิ่งที่เห็นบนหน้านี้

#### 📊 กล่องตัวเลขด้านบน (KPI Cards)
มี 4 กล่อง แสดงข้อมูลของ**วันนี้**:

| กล่อง | ความหมาย |
|------|----------|
| **ออเดอร์วันนี้** | จำนวนออเดอร์ที่รับมาวันนี้ทั้งหมด |
| **ยอดขายวันนี้** | มูลค่าสินค้าที่ขายได้วันนี้ (หน่วย: บาท) |
| **รอยืนยันออเดอร์** | ออเดอร์ที่ยังไม่ได้กด "ยืนยัน" — ต้องรีบดำเนินการ |
| **ยอดส่งสำเร็จเดือนนี้** | รวมยอดที่จัดส่งไปแล้วทั้งเดือน |

#### ✅ งานที่ต้องทำตอนนี้
มี 3 การ์ด บอกว่ามีงานค้างอยู่กี่อย่าง:

1. **รอยืนยันออเดอร์** — กดการ์ดนี้ → ไปหน้า "รายการออเดอร์"
2. **รอจัดส่ง** — กดการ์ดนี้ → ไปหน้า "ใบจัดส่ง"  
3. **จัดการสต็อก** — กดการ์ดนี้ → ไปหน้า "สต็อก"

#### 📈 แนวโน้มยอดขาย 7 วันล่าสุด
ตารางแสดงข้อมูลย้อนหลัง 7 วัน:
- วัน / จำนวนออเดอร์ / ยอดขาย / เปรียบเทียบวันก่อน (สีเขียว = เพิ่มขึ้น, สีแดง = ลดลง)

#### 📋 รายการล่าสุด
แสดงออเดอร์ล่าสุด 5-10 รายการ — กด **"ดูทั้งหมด"** เพื่อไปหน้ารายการออเดอร์ทั้งหมด

แต่ละแถวบอก: ชื่อร้านค้า / เลขออเดอร์ / วันที่ / สถานะ / ยอดรวม

**สีของสถานะ:**
- 🔵 **รับออเดอร์แล้ว** — ออเดอร์ที่ลูกค้าส่งมา ยังรอยืนยัน
- 🟢 **ยืนยันแล้ว** — ยืนยันและพร้อมจัดส่ง
- ⚫ **ฉบับร่าง** — ออเดอร์ที่สร้างไว้แต่ยังไม่ได้ส่ง
- 🔴 **ยกเลิก** — ออเดอร์ที่ถูกยกเลิกแล้ว

#### 🏆 ร้านค้ายอดขายสูงสุด 5 อันดับ
กราฟแท่งแสดงว่าร้านไหนสั่งเยอะที่สุดในช่วงนี้

#### 🥇 สินค้าขายดี 5 อันดับ
รูปและชื่อสินค้าที่ขายดีที่สุด พร้อมยอดขาย

#### ลิงก์ทางลัด (ปุ่มด้านล่าง)
- **รับออเดอร์** → ไปหน้า "รายการออเดอร์"
- **สรุปออเดอร์** → ไปหน้า "สรุปออเดอร์รายวัน"
- **ใบจัดส่ง** → ไปหน้า "ใบจัดส่ง"
- **ใบวางบิล** → ไปหน้า "ใบวางบิล"
- **สต็อก** → ไปหน้า "สต็อกสินค้า"
- **ร้านค้า** → ไปหน้า "ตั้งค่าร้านค้า"

## 4. รายการออเดอร์ขาเข้า

> **เมนู:** กดที่ "รายการออเดอร์" ในเมนูด้านข้าง  
> **สิทธิ์:** Admin เท่านั้น

### 📋 หน้านี้คืออะไร?
รายการออเดอร์ **ทุกใบ** ที่ลูกค้าส่งมาผ่านแอป LINE หน้านี้ใช้สำหรับ:
- ดูรายละเอียดออเดอร์แต่ละใบ
- แก้ไขจำนวนสินค้าในออเดอร์
- ยืนยันหรือยกเลิกออเดอร์
- สร้างออเดอร์ด้วยตนเอง (กรณีลูกค้าโทรมาสั่ง)

### 🔍 การกรองและค้นหา

**บนมือถือ** — กดปุ่มค้นหา (รูปแว่นขยาย) → จะเปิดหน้าต่างค้นหาขึ้นมา:
- **เลือกวันที่** — แตะเพื่อเลือกวันที่ต้องการดู (ค่าเริ่มต้น = วันนี้)
- **ช่องค้นหา** — พิมพ์เลขออเดอร์ / ชื่อร้าน → กด "ค้นหา"

**บนคอมพิวเตอร์** — ช่องกรอกอยู่ด้านบนของหน้า:
- เลือกวันที่จากปฏิทิน
- พิมพ์คำค้นหาในช่อง
- กดปุ่ม "ค้นหา"

### 📋 รายการออเดอร์ (การ์ดแต่ละใบ)

แต่ละออเดอร์จะแสดง:
- **เลขออเดอร์** เช่น TY-20250412-0001
- **ชื่อร้านค้า** ที่สั่ง
- **สถานะ** (สีป้ายบอก)
- **เวลา** ที่ส่งออเดอร์มา
- **ยอดรวม** (บาท)

### 👆 การกดดูรายละเอียดออเดอร์

**บนมือถือ** — กดที่การ์ดออเดอร์ → จะเปิดหน้าต่างแสดงรายละเอียด (modal):

หน้าต่างรายละเอียดออเดอร์แสดง:
- ชื่อร้านค้า / รหัสร้าน / เลขออเดอร์ / วันที่
- **ป้ายสถานะ** (สีตาม status)
- **ตารางสินค้า** แต่ละรายการ:
  - รูปสินค้า / ชื่อสินค้า / รหัสสินค้า
  - จำนวน / หน่วย / ราคาต่อหน่วย / รวม
  - ⚠️ ถ้ามีสัญลักษณ์ "ขาด X" (สีส้ม) หมายความว่าสต็อกไม่พอ

**ปุ่มในหน้าต่างรายละเอียด:**

| ปุ่ม | ทำอะไร | เงื่อนไข |
|-----|--------|---------|
| **◀ ▶** (ลูกศรซ้าย-ขวา) | เปลี่ยนดูออเดอร์ถัดไป/ก่อนหน้า | ใช้ได้ตลอด |
| **แก้ไขออเดอร์** | เปิดโหมดแก้ไขสินค้าในออเดอร์ | เฉพาะ "รับออเดอร์แล้ว" |
| **ยกเลิกออเดอร์** | ยกเลิกออเดอร์ทั้งใบ | เฉพาะ "รับออเดอร์แล้ว" |
| **✕** | ปิดหน้าต่าง | ใช้ได้ตลอด |

#### โหมดแก้ไขออเดอร์ (กดปุ่ม "แก้ไขออเดอร์")
เมื่อกดแล้ว สินค้าแต่ละรายการจะแสดงปุ่มควบคุม:
- **➕ (บวก)** — เพิ่มจำนวน
- **➖ (ลบ)** — ลดจำนวน (ถ้าลดเหลือ 0 สินค้ารายการนั้นจะถูกลบออก)
- **ลบ** — ลบรายการสินค้านั้นออกจากออเดอร์ทันที
- **บันทึก** — ยืนยันการแก้ไขทั้งหมด
- **ยกเลิก** — ไม่บันทึก กลับไปดูแบบเดิม

> ⚠️ **หมายเหตุ:** ถ้าลบสินค้าทุกรายการออกหมด ออเดอร์จะถูกยกเลิกอัตโนมัติ

#### การยกเลิกออเดอร์ (กดปุ่ม "ยกเลิกออเดอร์")
1. กดปุ่ม "ยกเลิกออเดอร์"
2. ระบบจะถามยืนยัน — กด "ยืนยัน" เพื่อดำเนินการ
3. ออเดอร์จะเปลี่ยนสถานะเป็น **"ยกเลิก"** (สีแดง)
4. สต็อกที่จองไว้จะถูกคืนกลับอัตโนมัติ

### ➕ การสร้างออเดอร์ด้วยตนเอง

ใช้เมื่อ: ลูกค้าโทรมาสั่งผ่านโทรศัพท์ ไม่ได้ส่งผ่านแอป LINE

**วิธีสร้าง:**
1. กดปุ่ม **"+ เพิ่มออเดอร์"** (ปุ่มสีน้ำเงิน มุมบนขวา หรือลอยด้านล่าง)
2. หน้าต่างสร้างออเดอร์เปิดขึ้น — กรอกข้อมูล:

| ช่อง | รายละเอียด | จำเป็น? |
|-----|-----------|--------|
| **ร้านค้า** | เลือกจากรายชื่อ dropdown | ✅ |
| **วันที่** | วันที่ต้องการให้ออเดอร์นี้ (ค่าเริ่มต้น = วันนี้) | ✅ |
| **หมายเหตุ** | จดบันทึกเพิ่มเติม (ไม่บังคับ) | ❌ |

3. กด **"+ เพิ่มสินค้า"** เพื่อเพิ่มสินค้าในออเดอร์:
   - เลือกชื่อสินค้าจาก dropdown
   - กรอกจำนวน
   - เลือกหน่วย (กก./แพ็ค/ถุง ฯลฯ)
   - ราคาจะแสดงอัตโนมัติ (ตามราคาที่ตั้งไว้)
   - กด **"+ เพิ่มรายการ"** เพื่อเพิ่มสินค้าอีกชิ้น

4. ตรวจสอบรายการทั้งหมด แล้วกด **"สร้างออเดอร์"**
5. ออเดอร์จะถูกสร้างและแสดงในรายการ

**ปุ่ม "คัดลอกจากเมื่อวาน"** (ถ้ามี):  
ถ้าร้านนี้เคยสั่งเมื่อวาน จะมีปุ่มให้กด → ระบบจะเติมรายการสินค้าเดิมจากเมื่อวานให้อัตโนมัติ ไม่ต้องกรอกใหม่

## 5. สรุปออเดอร์รายวัน

> **เมนู:** กดที่ "สรุปออเดอร์" ในเมนูด้านข้าง  
> **สิทธิ์:** Admin เท่านั้น

### 📊 หน้านี้คืออะไร?
แสดงออเดอร์ของแต่ละร้านในวันที่เลือก **รวมกันเป็นรายร้าน** เหมาะสำหรับดูภาพรวมว่าแต่ละร้านสั่งอะไรบ้าง

### สิ่งที่เห็น

**ส่วนกรองข้อมูล (ด้านบน):**
- เลือกวันที่ → ดูออเดอร์วันนั้น
- ค้นหาชื่อร้าน

**ตารางสรุปรายร้าน:**
- **ชื่อร้าน** / รหัสร้าน
- **จำนวนออเดอร์** ของร้านนั้นในวันที่เลือก
- **ยอดรวม** (บาท)

**กดที่แถวร้านค้า** → ขยายดูรายละเอียด:
- รายการสินค้าทั้งหมดที่ร้านนั้นสั่ง
- ปริมาณสต็อก (คงเหลือ / จองไว้)
- ถ้าสต็อกไม่พอ → จะมีสัญลักษณ์เตือน

## 6. ใบจัดส่ง

> **เมนู:** กดที่ "ใบจัดส่ง" ในเมนูด้านข้าง  
> **สิทธิ์:** Admin และ Warehouse

### 🚚 หน้านี้คืออะไร?
ใช้ติดตามและจัดการ**การส่งของ** ทราบว่าแต่ละร้านได้รับสินค้าครบหรือยัง และวางบิลแล้วหรือยัง

### สิ่งที่เห็นบนหน้านี้

#### 🔍 ส่วนกรองข้อมูล
- **จากวันที่** — วันเริ่มต้นที่ต้องการดู
- **ถึงวันที่** — วันสิ้นสุด
- **ช่องค้นหา** — พิมพ์ชื่อร้านหรือเลขออเดอร์
- กด **"ค้นหา"** หรือกด Enter

**บนมือถือ** — กดปุ่มค้นหา (รูปแว่น) มุมบนขวา → หน้าต่างค้นหาจะเปิดขึ้น

#### 📊 สรุปด้านบน (Summary Bar)
แสดงตัวเลขรวม:
- จำนวนร้านทั้งหมด
- จำนวนรายการสินค้า
- ยอดรวมที่สั่ง / ยอดที่ส่งจริง

#### 📋 รายการใบจัดส่งแต่ละร้าน

แต่ละการ์ดแสดง:
- **ชื่อร้านค้า** และรหัสร้าน
- **เลขออเดอร์** (รูปแบบ TY-YYYYMMDD-XXXX)
- **จำนวนรายการสินค้า**
- **ยอดที่สั่ง** vs **ยอดที่ส่งจริง**
- **ป้ายสถานะการวางบิล:**
  - 🟢 **"วางบิลแล้ว · เลขบิล"** — ออกใบวางบิลแล้ว
  - 🟡 **"ยังไม่วางบิล"** — ยังไม่ได้ออกบิล

#### 🖨️ ปุ่มพิมพ์ใบจัดส่ง
- **"พิมพ์ใบจัดส่ง"** (ปุ่มด้านบน) → เปิดหน้าพิมพ์ใบจัดส่งรวมทุกร้านของวันที่เลือก

#### 📦 ปุ่มพิมพ์ใบจัดของ
- **"พิมพ์ใบจัดของ"** → เปิดหน้า matrix สินค้า × ร้านค้า สำหรับจัดของก่อนขึ้นรถ

### 🖊️ การแก้ไขจำนวนที่ส่งจริง (สำหรับ Admin)

เมื่อกดเข้าดูรายละเอียดใบจัดส่งแต่ละใบ:

1. กดปุ่ม **"แก้ไข"** (ปุ่มดินสอ)
2. ระบบเปิดโหมดแก้ไข — แต่ละสินค้าจะมีช่องกรอกตัวเลข
3. แก้ไขจำนวนที่ส่งจริง (สามารถลดได้ แต่ไม่สามารถเพิ่มเกินที่สั่งมา)
4. กด **"บันทึก"** → ยืนยันการแก้ไข
5. กด **"ยกเลิก"** → ไม่บันทึก กลับไปดูแบบเดิม

> **ใช้เมื่อ:** ส่งสินค้าไปน้อยกว่าที่สั่ง เช่น สินค้าขาด หรือส่งผิด

### 🖨️ หน้าพิมพ์ใบจัดส่ง (ปุ่ม "พิมพ์ใบจัดส่ง")
เปิดในแท็บใหม่ แสดง:
- หัวกระดาษ: ชื่อบริษัท, วันที่
- รายการสินค้าทุกร้านพร้อมรูปภาพ
- กด **"พิมพ์"** → โทรศัพท์/คอมพิวเตอร์จะเปิดเมนูพิมพ์
- กด **"กลับ"** → ปิดหน้าพิมพ์

### 📦 หน้าใบจัดของ (/orders/packing-list)
ตารางสินค้า × ร้านค้า — ดูว่าต้องหยิบสินค้าอะไรเท่าไหร่ส่งให้ร้านไหน:
- **แถว (แนวนอน):** ชื่อสินค้า / รหัส / หน่วย
- **คอลัมน์ (แนวตั้ง):** ชื่อร้านค้าแต่ละร้าน
- **ตัวเลขในตาราง:** จำนวนที่ต้องส่ง
- ถ้าร้านไหนไม่มีรถประจำ → จะมีข้อความเตือน
- กด **"พิมพ์ใบจัดของ"** เพื่อพิมพ์

## 7. ใบวางบิล

> **เมนู:** กดที่ "ใบวางบิล" ในเมนูด้านข้าง  
> **สิทธิ์:** Admin เท่านั้น

### 💰 หน้านี้คืออะไร?
ใช้ **สร้างและพิมพ์ใบวางบิล** ให้ลูกค้า และดูประวัติการวางบิลที่ผ่านมา

### ส่วนที่ 1: ออกใบวางบิลใหม่

**กรอกข้อมูล:**

| ช่อง | รายละเอียด |
|-----|-----------|
| **ร้านค้า** | เลือกร้านที่ต้องการออกบิล หรือเลือก "ทุกร้านที่มีรายการในช่วงนี้" เพื่อออกบิลรวมทุกร้านพร้อมกัน |
| **ตั้งแต่วันที่** | วันเริ่มต้นของรอบบิล (เลือกจากปฏิทิน) |
| **ถึงวันที่** | วันสิ้นสุดของรอบบิล |

**ปุ่มดำเนินการ:**
- **"ดูและพิมพ์ใบวางบิล"** — เปิดตัวอย่างใบวางบิลสำหรับร้านที่เลือก (เปิดแท็บใหม่)
- **"ดูและพิมพ์ใบวางบิลทุกร้าน"** — เปิดตัวอย่างสำหรับทุกร้านพร้อมกัน

> 📌 **หมายเหตุ:** ใบวางบิลจะแสดงเฉพาะออเดอร์ที่ **"ยืนยันแล้ว"** และ **"จัดส่งแล้ว"** ในช่วงวันที่เลือก

### หน้าพิมพ์ใบวางบิล (/billing/print)
เปิดในแท็บใหม่:
- แสดงใบวางบิลฉบับสมบูรณ์พร้อมพิมพ์
- **"บันทึกและพิมพ์"** — บันทึกบิลนี้ลงระบบ (ล็อกยอด) แล้วพิมพ์
- **"พิมพ์"** — พิมพ์โดยไม่บันทึก
- **"กลับ"** — ปิดหน้าพิมพ์

> ⚠️ **สำคัญ:** เมื่อกด "บันทึกและพิมพ์" แล้ว ยอดบิลจะ**ล็อก**ไม่สามารถแก้ไขได้อีก

### ส่วนที่ 2: ประวัติการวางบิล

แสดงรายการบิลที่เคยออกไปแล้ว เรียงจากใหม่ไปเก่า

**แต่ละรายการบิลแสดง:**
- 🔒 **เลขบิล** (รูปแบบ: ตัวเลข/ปีพ.ศ.) + ป้าย "ยอดล็อก"
- **ชื่อร้าน** / **รหัสร้าน**
- **ช่วงวันที่** ของบิล
- **ยอดรวม** (บาท)
- รายการใบส่งของที่รวมอยู่ในบิลนี้
- ปุ่ม **"พิมพ์อีกครั้ง"** → เปิดใบวางบิลเดิมพิมพ์ซ้ำได้

**ถ้ายังไม่มีประวัติ:**  
จะเห็นข้อความ "ยังไม่มีประวัติการวางบิล — ใบวางบิลที่กด 'บันทึกและพิมพ์' จะแสดงที่นี่"

## 8. สต็อกสินค้า

> **เมนู:** กดที่ "สต็อก" ในเมนูด้านข้าง  
> **สิทธิ์:** Admin เท่านั้น

### 📦 หน้านี้คืออะไร?
ดูปริมาณสินค้าที่มีในคลัง และ **รับสินค้าเข้าคลัง** เมื่อสั่งซื้อจากโรงงาน/ซัพพลายเออร์

### แท็บ (Tabs)
- **"สต็อก"** — หน้าปัจจุบัน (ดูสต็อกคงเหลือ)
- **"ความเคลื่อนไหวสต็อก"** — ดูประวัติการเข้า-ออกของสินค้า

### 📊 กล่องสรุปด้านบน (Summary Cards)
| กล่อง | ความหมาย |
|------|----------|
| มูลค่าสต็อกทั้งหมด | ราคารวมของสินค้าที่มีในคลัง |
| จองไว้ | จำนวน/มูลค่าที่ถูกจองในออเดอร์ที่รอส่ง |
| ต้นทุนรวม | ราคาทุนของสินค้าในคลัง |
| จำนวนสินค้า | จำนวนรายการสินค้าทั้งหมด |

### 📋 ตารางสต็อกสินค้า
แต่ละแถวแสดง:
- **รูปสินค้า**
- **รหัสสินค้า (SKU)** เช่น TYN-001
- **ชื่อสินค้า**
- **สต็อกคงเหลือ** (จำนวน + หน่วย)
- **จองไว้** (ออเดอร์ที่รอส่ง)
- **หมวดหมู่**
- **ราคาต้นทุน** (ต่อหน่วย)
- **มูลค่ารวม** (คงเหลือ × ต้นทุน)

### ➕ การรับสินค้าเข้าคลัง

**วิธีที่ 1: กดปุ่มลอย**  
กดปุ่ม **"+ รับสินค้า"** (ลอยอยู่มุมล่างขวา) หรือ **"+ เพิ่มสต็อก"**

**วิธีที่ 2: ผ่านเมนูตั้งค่า**  
ไปที่ ตั้งค่า → รับสต็อก

**หน้าต่างรับสินค้า (modal) จะเปิดขึ้น:**

กรอกข้อมูล:
1. **สินค้า** — เลือกชื่อสินค้าจาก dropdown
2. **จำนวน** — กรอกตัวเลขที่รับเข้า
3. **หมายเหตุ** — บันทึกข้อมูลเพิ่มเติม (ไม่บังคับ) เช่น "รับจากโรงงาน ล็อต A"

กด **"บันทึก"** → สต็อกจะเพิ่มขึ้นทันที  
กด **"ยกเลิก"** → ปิดหน้าต่างโดยไม่บันทึก

## 9. ความเคลื่อนไหวสต็อก

> **เมนู:** กดที่ "สต็อก" → แล้วกดแท็บ "ความเคลื่อนไหวสต็อก"  
> **สิทธิ์:** Admin เท่านั้น

### 📜 หน้านี้คืออะไร?
บันทึกประวัติ **ทุกครั้งที่สต็อกมีการเปลี่ยนแปลง** ไม่ว่าจะเพิ่มหรือลด

### ตารางความเคลื่อนไหว
แต่ละแถวแสดง:

| คอลัมน์ | ความหมาย |
|--------|----------|
| **วันที่/เวลา** | เมื่อไหร่ที่มีการเปลี่ยนแปลง |
| **ประเภท** | รับเข้า / จองไว้ / ส่งออก / ปรับแก้ |
| **สินค้า** | ชื่อสินค้าที่เปลี่ยนแปลง |
| **จำนวน** | ±ตัวเลข (บวก=เพิ่ม, ลบ=ลด) |
| **สต็อกก่อน** | จำนวนก่อนเปลี่ยน |
| **สต็อกหลัง** | จำนวนหลังเปลี่ยน |
| **หมายเหตุ** | เหตุผล/บันทึก |
| **ผู้ดำเนิน** | ใครทำรายการ |

### ประเภทการเคลื่อนไหว
- **รับเข้า (receipt)** — รับสินค้าเข้าคลัง (บวก)
- **จองไว้ (reserve)** — ออเดอร์ถูกยืนยัน ระบบจองสต็อก (ลด)
- **ส่งออก (issue)** — ส่งสินค้าไปแล้ว (ลด)
- **คืน (release)** — ยกเลิกออเดอร์ คืนสต็อก (บวก)
- **ปรับแก้ (adjustment)** — แก้ไขด้วยตนเอง

## 10. ตั้งค่าร้านค้า

> **เมนู:** ตั้งค่า → จัดการร้านค้า  
> **สิทธิ์:** Admin เท่านั้น

### 🏪 หน้านี้คืออะไร?
จัดการข้อมูล**ร้านค้าที่เป็นลูกค้า** — เพิ่ม แก้ไข ลบ และกำหนดรถประจำ

### แท็บ
- **"ร้านค้า"** — จัดการข้อมูลร้าน (หน้าปัจจุบัน)
- **"ผูกราคา"** — ตั้งราคาขายเฉพาะรายร้าน (ดูส่วนถัดไป)

### รายการร้านค้า
แสดงตารางร้านทั้งหมด แต่ละแถวมี:
- **รหัสร้าน** (TYS-001, TYS-002, ...)
- **ชื่อร้านค้า**
- **ที่อยู่**
- **เบอร์โทรศัพท์**
- **รถประจำ**
- ปุ่ม **"แก้ไข"** (รูปดินสอ)
- ปุ่ม **"ลบ"** (รูปถังขยะ)

### ➕ เพิ่มร้านค้าใหม่

กดปุ่ม **"+ เพิ่มร้านค้า"** → หน้าต่างเพิ่มร้านค้าเปิดขึ้น:

| ช่อง | รายละเอียด | จำเป็น? |
|-----|-----------|--------|
| **รหัสร้านค้า** | ระบบสร้างให้อัตโนมัติ (TYS-XXX) แก้ไขได้ | ✅ |
| **ชื่อร้านค้า** | ชื่อร้านค้า/บุคคล | ✅ |
| **ที่อยู่** | ที่อยู่สำหรับพิมพ์ลงใบจัดส่ง | ❌ |
| **เบอร์โทรศัพท์** | เบอร์ติดต่อร้าน | ❌ |
| **รถประจำ** | เลือกรถที่ส่งให้ร้านนี้ประจำ | ❌ |

กด **"บันทึก"** → ร้านค้าใหม่จะปรากฏในรายการ  
กด **"ยกเลิก"** → ปิดหน้าต่างโดยไม่บันทึก

### ✏️ แก้ไขข้อมูลร้านค้า
1. กดปุ่ม "แก้ไข" ข้างชื่อร้าน → หน้าต่างแก้ไขเปิดขึ้น (ข้อมูลเดิมถูกกรอกมาแล้ว)
2. แก้ไขข้อมูลที่ต้องการ
3. กด **"บันทึก"**

### 🗑️ ลบร้านค้า
1. กดปุ่ม "ลบ" (สีแดง) ข้างชื่อร้าน
2. ระบบจะถามยืนยัน — กด "ยืนยัน"

> ⚠️ **ระวัง:** การลบร้านค้าไม่สามารถเรียกคืนได้

## 11. ผูกราคาสินค้าตามร้านค้า

> **เมนู:** ตั้งค่า → จัดการร้านค้า → แท็บ "ผูกราคา"  
> **สิทธิ์:** Admin เท่านั้น

### 💲 หน้านี้คืออะไร?
ตั้งราคาขายสินค้า **แตกต่างกันตามแต่ละร้าน** เช่น ร้าน A ซื้อเส้นใหญ่ราคา 35 บาท/กก. แต่ร้าน B ซื้อที่ 33 บาท/กก.

### วิธีใช้งาน

1. **เลือกร้านค้า** จากรายการด้านซ้าย → คลิกที่ชื่อร้าน
2. ด้านขวาจะแสดงราคาสินค้าที่กำหนดไว้สำหรับร้านนั้น
3. ถ้ายังไม่มี → จะเห็นข้อความ "ยังไม่มีราคาพิเศษ"

### ➕ เพิ่มราคาพิเศษ

1. เลือกร้านค้าจากรายการ
2. กดปุ่ม **"+ เพิ่มราคา"**
3. หน้าต่างเพิ่มราคาเปิดขึ้น:
   - เลือก **สินค้า** จาก dropdown (พร้อมรูปภาพ)
   - เลือก **หน่วย** (กก./แพ็ค ฯลฯ)
   - กรอก **ราคา** (บาท)
4. กด **"บันทึก"**

### ✏️ แก้ไขราคา
- กดปุ่ม "แก้ไข" ข้างรายการราคา → แก้ไขตัวเลข → บันทึก

### 🗑️ ลบราคาพิเศษ
- กดปุ่ม "ลบ" → ราคาพิเศษถูกลบ ระบบจะใช้ราคาปกติแทน

> **หมายเหตุ:** ถ้าไม่ได้ตั้งราคาพิเศษ ระบบจะใช้ราคาขายมาตรฐานของสินค้านั้น

## 12. ตั้งค่าสินค้า

> **เมนู:** ตั้งค่า → จัดการสินค้า  
> **สิทธิ์:** Admin เท่านั้น

### 🛒 หน้านี้คืออะไร?
จัดการ**รายการสินค้าทั้งหมด** — เพิ่มสินค้าใหม่ แก้ไขข้อมูล อัปโหลดรูป และจัดหมวดหมู่

### แท็บ
- **"จัดการสินค้า"** — รายการสินค้า (หน้าปัจจุบัน)
- **"เพิ่มหมวดหมู่"** — จัดการหมวดหมู่สินค้า

### รายการสินค้า
แต่ละแถวแสดง:
- **รูปสินค้า**
- **รหัสสินค้า (SKU)** เช่น TYN-001

### ➕ เพิ่มสินค้าใหม่

กดปุ่ม **"+ เพิ่มสินค้า"** → หน้าต่างเพิ่มสินค้าเปิดขึ้น:

**ข้อมูลพื้นฐาน:**

| ช่อง | รายละเอียด | จำเป็น? |
|-----|-----------|--------|
| **รหัสสินค้า (SKU)** | ระบบสร้างให้อัตโนมัติ (TYN-XXX) | ✅ |

| **ราคาต้นทุน** | ต้นทุนต่อหน่วยหลัก (บาท) | ✅ |

**อัปโหลดรูปสินค้า:**
- กดปุ่ม **"เลือกรูปภาพ"** → เลือกรูปจากอัลบั้มในมือถือ
- สามารถอัปโหลดได้หลายรูป
- รูปแรกจะเป็นรูปหลัก

**หน่วยขาย (Sale Units):**

- ขาย "แบบถุง 5 กก." ราคา 170 บาท/ถุง

กด **"+ เพิ่มหน่วยขาย"** เพื่อเพิ่ม:
- ชื่อหน่วย เช่น "ถุง 5 กก."
- สัดส่วน (เทียบกับหน่วยหลัก) เช่น 5
- ราคาขาย
- โหมดต้นทุน: "ดึงจากฐาน" (คำนวณอัตโนมัติ) หรือ "กำหนดเอง"

กด **"บันทึก"** → สินค้าถูกบันทึก  
กด **"ยกเลิก"** → ปิดหน้าต่าง

ใช้จัดกลุ่มสินค้า เช่น "เส้นก๋วยเตี๋ยว", "น้ำซุป", "เครื่องเคียง"

**เพิ่มหมวดหมู่:**
1. กรอกชื่อหมวดหมู่
2. กำหนดลำดับการแสดง (หมวดที่ 1 อยู่บนสุด)
3. กด "บันทึก"

**แก้ไข/ลบหมวดหมู่:**
- กดปุ่มดินสอ (แก้ไข) หรือถังขยะ (ลบ) ข้างชื่อหมวดหมู่

## 13. ตั้งค่ารถส่งของ

> **เมนู:** ตั้งค่า → จัดการรถ  
> **สิทธิ์:** Admin เท่านั้น

### 🚛 หน้านี้คืออะไร?
จัดการข้อมูล**รถที่ใช้ส่งของ** — เพิ่มรถ แก้ไข และผูกรถกับร้านค้าแต่ละร้าน

### รายการรถ
แต่ละแถวแสดง:
- **ชื่อรถ** เช่น "รถกระบะเบนซ์"
- **ป้ายทะเบียน** เช่น "กก-1234 กรุงเทพ"
- **จำนวนร้านค้าที่ผูก** — ร้านไหนบ้างที่ใช้รถคันนี้ส่ง
- ปุ่ม **"แก้ไข"** / **"ลบ"**

### ➕ เพิ่มรถใหม่

กดปุ่ม **"+ เพิ่มรถ"** → หน้าต่างเพิ่มรถเปิดขึ้น:

| ช่อง | รายละเอียด |
|-----|-----------|
| **ชื่อรถ** | ระบุชื่อเรียกรถ เช่น "รถส่งสาย 1" |
| **ป้ายทะเบียน** | เลขทะเบียนรถ |

กด **"บันทึก"** → รถใหม่ปรากฏในรายการ

> **หมายเหตุ:** หลังเพิ่มรถแล้ว ต้องไปผูกกับร้านค้าที่หน้า "จัดการร้านค้า" (ช่อง "รถประจำ")

## 14. รายงานภาพรวมยอดขายรายปี

> **เมนู:** รายงาน → ภาพรวมยอดขายรายปี  
> **สิทธิ์:** Admin และ Member

### 📈 หน้านี้คืออะไร?
ดู**สรุปยอดขายรายเดือนตลอดทั้งปี** กำไร-ขาดทุน และแนวโน้ม

### การเลือกปี
มีปุ่มตัวเลขปี พ.ศ. ให้เลือก — กดเพื่อเปลี่ยนดูปีอื่น

### 📊 กล่องสรุป (KPI)
| กล่อง | ความหมาย |
|------|----------|
| **ยอดขายรวมทั้งปี** | รายได้รวมตลอดปีที่เลือก |
| **กำไรสุทธิ** | กำไรหลังหักต้นทุน (พร้อม % กำไร) |
| **เดือนยอดดีที่สุด** | เดือนที่มียอดขายสูงสุดในปีนั้น |
| **เฉลี่ย/เดือน** | ยอดขายเฉลี่ยต่อเดือน พร้อมจำนวนออเดอร์ |

### 📉 กราฟและตาราง
- กราฟแท่งยอดขายรายเดือน (มกราคม - ธันวาคม)
- กราฟจำนวนออเดอร์ต่อเดือน
- แนวโน้มกำไร (เส้นกราฟ)
- สรุปกำไร: "ปี 2568 มีกำไรสุทธิ XX,XXX บาท คิดเป็น XX% ของยอดขายรวม"

**ถ้าไม่มีข้อมูล:**  
จะเห็น "ไม่มีข้อมูลยอดขายในปี พ.ศ. XXXX — ลองเลือกปีอื่น"

## 15. รายงานยอดขายตามสินค้า

> **เมนู:** รายงาน → ยอดขายสินค้า  
> **สิทธิ์:** Admin และ Member

### 🏅 หน้านี้คืออะไร?
อันดับสินค้าที่ขายดีและทำกำไรมากที่สุด สามารถกรองตามช่วงเวลาและร้านค้าได้

### 🔍 การกรองข้อมูล

**บนมือถือ:** กดปุ่มค้นหา (รูปแว่น) → หน้าต่างกรองเปิดขึ้น:
- **สินค้า** — เลือกสินค้าที่ต้องการดู (เลือกหลายชิ้นได้)
- **ร้านค้า** — กรองเฉพาะร้านที่ต้องการ
- **ตั้งแต่วันที่** / **ถึงวันที่** — ช่วงเวลา
- กด **"ค้นหา"**

### 📊 กล่องสรุป
| กล่อง | ความหมาย |
|------|----------|
| ยอดขายรวม | รายได้รวมจากช่วงที่เลือก |
| จำนวน | จำนวนสินค้าที่ขายได้ทั้งหมด |
| ต้นทุนรวม | ต้นทุนสินค้าที่ขายไป |
| กำไรสุทธิ | กำไรหลังหักต้นทุน (สีแดง = ขาดทุน) |

### 🏆 ตารางอันดับสินค้า
แต่ละแถวแสดง:
- **อันดับ** 🥇🥈🥉
- **รหัสสินค้า**
- **ชื่อสินค้า** (พร้อมรูปภาพเล็ก)
- **จำนวนที่ขาย** / หน่วย
- **ต้นทุน** / **ยอดขาย** / **กำไรสุทธิ** / **กำไร%**
  - สีเขียว = กำไร
  - สีแดง = ขาดทุน

### การเลื่อนหน้า (Pagination)
- กดปุ่ม **"◀ ก่อนหน้า"** / **"ถัดไป ▶"** เพื่อดูหน้าอื่น
- แสดง 25 รายการต่อหน้า

### 🖨️ การพิมพ์รายงาน
กดปุ่ม **"พิมพ์"** มุมบนขวา → เปิดรายงานฉบับพิมพ์ (แสดงทุกรายการพร้อมโลโก้บริษัท)

## 17. รายงานใบจัดส่งรายวัน

> **เมนู:** รายงาน → รายงานใบจัดส่ง  
> **สิทธิ์:** Admin และ Member

### 📄 หน้านี้คืออะไร?
ดูประวัติ**ใบจัดส่งทั้งหมด** พร้อมมูลค่าและกำไร สามารถค้นหาย้อนหลังได้

### 🔍 การกรองข้อมูล
- เลือกร้านค้า / ช่วงวันที่ → กด "ค้นหา"

### 📊 กล่องสรุป
| กล่อง | ความหมาย |
|------|----------|
| รวมจำนวนใบส่ง | จำนวนใบจัดส่งทั้งหมดในช่วงที่เลือก |
| จำนวนรายการ | จำนวนบรรทัดสินค้า |
| ยอดรวมจัดส่ง | มูลค่ารวม (บาท) |
| จำนวนร้านค้า | ร้านที่มีการส่งในช่วงนี้ |

### ตารางรายการ
**จัดกลุ่มตามวัน** — แต่ละวันแสดง:
- หัวกลุ่ม: วันที่ + ยอดรวมของวันนั้น
- แต่ละใบจัดส่ง: เลขที่ / ร้าน / สินค้า / จำนวน / มูลค่า / กำไร

### 🖨️ การพิมพ์
กดปุ่ม **"พิมพ์"** → เปิดรายงานฉบับพร้อมพิมพ์

## 18. ขั้นตอนการทำงานตั้งแต่ต้นจนจบ

### 🔄 วงจรการทำงานของระบบ (ทำตามลำดับ)

```
1️⃣  ลูกค้าสั่งผ่าน LINE
         ↓
2️⃣  ออเดอร์เข้าระบบ (สถานะ: รับออเดอร์แล้ว 🔵)
         ↓
3️⃣  Admin ตรวจและยืนยัน
    → หน้า "รายการออเดอร์"
    → ดูรายละเอียด → ถ้าถูกต้องก็รอระบบประมวลผล
    → กรณีมีปัญหา: แก้ไขจำนวน หรือยกเลิก
         ↓
4️⃣  ออเดอร์ถูกยืนยัน (สถานะ: ยืนยันแล้ว 🟢)
    ระบบจอง stock อัตโนมัติ
         ↓
5️⃣  ก่อนออกรถ: พิมพ์ใบจัดของ
    → หน้า "ใบจัดส่ง" → กด "พิมพ์ใบจัดของ"
    → จัดสินค้าตามตาราง (สินค้า × ร้าน)
         ↓
6️⃣  ระหว่างส่ง: พิมพ์ใบจัดส่ง
    → หน้า "ใบจัดส่ง" → กด "พิมพ์ใบจัดส่ง"
    → พนักงานขับรถพาใบนี้ไปด้วย
         ↓
7️⃣  หลังส่ง: แก้ไขจำนวนจริง (ถ้าจำเป็น)
    → หน้า "ใบจัดส่ง" → กดใบของร้านนั้น → กด "แก้ไข"
    → ใส่จำนวนที่ส่งจริง → บันทึก
         ↓
8️⃣  วางบิลรายเดือน
    → หน้า "ใบวางบิล"
    → เลือกร้าน + ช่วงวันที่ → กด "ดูและพิมพ์"
    → ตรวจสอบ → กด "บันทึกและพิมพ์" (ล็อกยอด)
         ↓
9️⃣  ดูรายงาน
    → หน้า "รายงาน" ต่างๆ เพื่อวิเคราะห์ธุรกิจ
```
